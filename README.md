# Admin Dashboard

Admin Dashboard adalah project **ReactJS** berbasis Tailwind CSS yang menjadi template/dashboard UI interaktif.  
Dashboard ini menampilkan berbagai komponen seperti **sidebar navigasi**, **theme toggle (dark/light mode)**, **charts**, **notification**, **chat panel**, dan **layanan chart kompleks menggunakan Syncfusion React Charts**.
